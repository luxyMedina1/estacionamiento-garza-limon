# Sistema de Administración de Estacionamiento

Aplicación web desarrollada para administrar de forma manual el acceso, control de estancias y cobro de vehículos en un estacionamiento utilizando una arquitectura **MVC (Modelo-Vista-Controlador)**.

## Tecnologías Utilizadas

- **Backend:** Node.js, Express
- **Base de Datos:** MySQL (Relacional)
- **Vistas (Interfaz):** EJS con estilos personalizados
- **Librerías de Reportes:** ExcelJS, PDFKit

## Requisitos Previos

- [Node.js](https://nodejs.org/) (Versión v18 o superior recomendada)
- Servidor MySQL local activo (XAMPP, Wampserver o MySQL Server)

## Instalación y Configuración
